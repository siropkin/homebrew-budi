# Homebrew Tap for Budi

AI cost analytics for coding agents — know where your tokens and money go.

## Installation

```bash
brew install siropkin/budi/budi
```

## Usage

```bash
budi init    # Install hooks for Claude Code & Cursor
budi sync    # Sync data and start daemon
budi status  # Show cost summary in statusline
```

For more information, visit [github.com/siropkin/budi](https://github.com/siropkin/budi).
